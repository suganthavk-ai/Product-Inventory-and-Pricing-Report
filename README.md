# Product-Inventory-and-Pricing-Report
To manage and analyze the product sales, pricing, and inventory data efficiently using Excel.
# Description
The Product Inventory and Pricing Report is an Excel-based project designed to analyze product inventory, pricing, and category distribution across different countries and months. 

# Project Objective
 The project aims to:
  •	Analyze product pricing 
  •	Categorize products by type 
  •	Identify high-price products 

# Getting Started
## Dependencies
 The following tools and software were used in this project:
  • OS Version
  • MS Office latest version
  • Knowledge of Excel formula

# Dataset Description
|Column Name | Description|
|----|---------|
|Product ID|Unique identifier assigned to each product|
|Product Name|Name of the product|
|Brand Name|Brand or manufacturer name|
|Price ($)|Price of the product in USD|
|Quantity |Number of units available or sold|
|Category|Product category|
|Price Range	|High Price or Standard Price classification|
|Day|Day of the record|
|Country code|Country abbreviation|
|Month | Month of the record|

# Installation
1. Download the dataset file.
2. Save a copy of the file on your PC or laptop.
3. Open the file using Microsoft Excel.
4. If a Protected View message appears, click Enable Editing.
5. Review the dataset and explore the product inventory and pricing details.
6. Use Excel features such as:
  * Sorting
  * Filtering
  * Formulas
  * Tables
  * Conditional Formatting
7. Run the provided Excel formula to classify products into:
 * High Price
 * Standard Price

# Executing program
## Step to run the program
• Open the Product Inventory and Pricing Report Excel file.
• Ensure that editing is enabled in Microsoft Excel.
• Verify that all dataset columns are properly formatted.
